# M5Dial-Badge

丸い画面のついた小さな機械「M5Dial」を、**自分だけの電子バッヂ**にするプロジェクトです。

好きな写真や短い動画を入れておくと、丸い画面でスライドショーのように次々と映してくれます。写真の入れ替えは **スマホだけ** ででき、パソコンや専用アプリ、インターネット接続はいりません。

![デフォルトで入っている画像](assets/builtin.png)

---

## これは何ができるの?

- 胸元や机に置いて、好きな画像・動画をループ表示する電子バッヂになります
- 表示する写真はスマホのカメラで QR コードを読むだけで差し替えられます
- 電波の届く範囲で、その場で完結します(クラウドや会員登録は不要)

---

## 使い方(スマホで写真を入れる)

機械の画面に出る QR コードを **2 回** 読むだけです。順番に説明します。

1. **電源を入れる** … しばらくするとスライドショーが始まります
2. **画面下のボタンを 2 秒ほど長押し** … QR コードが表示されます
3. **① Wi-Fi の QR** をスマホのカメラで読む
   → 「`M5Dial-Badge` に接続しますか?」と出るのでタップ(**パスワードはありません**)
4. **ダイヤル(画面のふち)を回す** と QR が切り替わります。**② URL の QR** を読む
   → スマホのブラウザで操作画面が開きます
5. 操作画面で **写真を選ぶ → アップロード** … 数秒で機械の画面に反映されます

> 💡 QR は 2 種類あります。①でバッヂの Wi-Fi につなぎ、②で操作画面を開く、という流れです。

### 機械本体での操作

| 操作 | できること |
| --- | --- |
| ダイヤルを回す | 前の写真 / 次の写真に切り替え |
| 画面をタッチ | 一時停止 / 再開 |
| ボタンを短く押す | 状態表示(つなぎ方・IP アドレス・空き容量) |
| ボタンを 2 秒長押し | QR コードを表示 |

### スマホの操作画面でできること

- 保存した写真の一覧表示・並べ替え(▲▼)・削除・**全削除**
- 写真ごとの表示時間(秒)の変更
- 「表示」ボタンで、その写真をすぐ画面に出す
- 空き容量の確認

> 自宅の Wi-Fi につなぐ設定もできます(後のバージョンで対応予定)。つながらないときは自動でバッヂ自身の Wi-Fi に戻ります。

---

## うまくいかないときは

| こまったこと | 対処 |
| --- | --- |
| QR を読んでも Wi-Fi につながらない | スマホの Wi-Fi 設定を開き、手動で `M5Dial-Badge` を選んでみてください |
| つないだのに操作画面が開かない | ブラウザで `http://192.168.4.1/` を直接入力してみてください |
| 「インターネット未接続」と出る | 正常です。そのまま「このまま接続」を選んでください |
| 画面が真っ暗・反応しない | いったん電源を入れ直してください |

---

## 対応している写真・動画

- **写真**: スマホの普通の写真(JPEG/PNG など)。丸い画面に合わせて自動で正方形に切り取られます
- **動画**: スマホで撮った動画も使えます。ブラウザ内で自動的にパラパラ漫画(JPEG連番)に変換して取り込み、丸い画面でループ再生します(fps・画質・長さを調整可)

## 開発者向け情報

<details>
<summary>プログラムをビルド・書き込みする方はこちら(クリックで開く)</summary>

> 🔰 **はじめての方へ**: クローンから Claude Code で開発を続ける手順を
> [docs/GETTING_STARTED.md](docs/GETTING_STARTED.md) にステップバイステップでまとめています。

[PlatformIO](https://platformio.org/) + Arduino フレームワークで開発しています。

```bash
# ビルド
pio run -e m5dial

# 書き込み(ポートは ls /dev/cu.usbmodem* で確認)
pio run -e m5dial -t upload --upload-port /dev/cu.usbmodem101

# シリアルモニタ(動作ログの確認)
pio device monitor -b 115200
```

Web 操作画面を変更したら、`python3 tools/build_webui.py` で `src/web_ui.h` を再生成します
(ビルド時に自動実行されます)。
